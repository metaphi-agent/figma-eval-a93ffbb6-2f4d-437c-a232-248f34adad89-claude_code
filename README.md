# 20+ Modals, Popups & Alerts

A comprehensive React component library featuring over 20 modal, popup, and alert designs for modern UI applications.

## Features

- ✅ Alert components with and without icons
- ✅ Project success alerts with Learn More links
- ✅ Toast notifications with undo buttons
- ✅ Popups for accepting changes, verifying identity, and terms agreements
- ✅ Modals with checkbox variants and image support
- ✅ Newsletter subscription modals

## Tech Stack

- React 19
- TypeScript
- Tailwind CSS v4
- Vite

## Getting Started

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```
