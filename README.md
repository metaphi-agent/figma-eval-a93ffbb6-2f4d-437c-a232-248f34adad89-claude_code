# 20+ Modals, Popups & Alerts

A comprehensive collection of modal and alert components built with React and Tailwind CSS v4.

## Components

### UI Components
- `Button` - Primary, secondary, and ghost variants
- `Input` - With label, error state, and password toggle
- `Checkbox` - With label support
- `CloseButton` - Circular close button

### Alert Components
- `Alert` - With icon, title, description
- `ProjectAlert` - Simple text alert
- `AlertBar` - Horizontal alert with close button
- `AlertBarUndo` - Horizontal alert with undo button

### Toast Components
- `Toast` - With icon, message, undo/close options
- `CheckIcon` - Success checkmark icon

### Popup Components
- `AcceptChangesPopup` - Confirmation with icon
- `VerifyIdentityPopup` - Password verification
- `TermsPopup` - Terms and agreements

### Modal Components
- `ConfirmModal` - Confirmation with icon
- `SuccessModal` - Success state
- `SuccessUndoModal` - Success with undo option
- `AcceptModal` - Simple accept/reject
- `AcceptCheckboxModal` - With centered checkbox
- `AcceptCheckboxLeftModal` - With left-aligned checkbox
- `AcceptImageModal` - With image header
- `NewsletterModal` - Email subscription
- `NewsletterImageModal` - Email subscription with image

## Getting Started

```bash
npm install
npm run dev
```
